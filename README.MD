# Spring Animation

## Purpose
Building an animation to celebrate Spring with the assistance of Claude. It includes the source logic and platform-optimized assets for iOS, Android, and Web integration.

## Project Structure
* `/assets`: Raw source files and project files.
* `/dist`: Production-ready exports organized by platform.
* `/drafts`: Experimental versions and discarded iterations.

## Platform Exports
| Platform | Format | Status |
| :--- | :--- | :--- |
| **iOS** | JSON / Swift | Ready |
| **Android** | JSON / XML | Ready |
| **Web** | GIF / WebM | Ready |
